# Angry (غاضب)
- ما هذا الهراء؟ (Ma hatha al-hura'?) - What is this nonsense?
- أنا غاضب جدا! (Ana ghadib jiddan!) - I am very angry!
- لا تتحدث معي هكذا! (La tatahaddath ma'i hakatha!) - Don't talk to me like that!

# Fearful (خائف)
- أنا خائف... (Ana kha'if...) - I'm scared...
- هل سمعت هذا الصوت؟ (Hal sami'ta hatha al-sawt?) - Did you hear that sound?
- أرجوك ساعدني! (Arjook sa'idni!) - Please help me!

# Happy (سعيد)
- يا له من يوم رائع! (Ya lahu min yawm ra'i'!) - What a wonderful day!
- أنا سعيد جداً لرؤيتك. (Ana sa'eed jiddan li-ru'yatik.) - I'm so happy to see you.
- هذا مضحك جدا! (Hatha mudhik jiddan!) - This is very funny!

# Neutral (محايد)
- الكتاب على الطاولة. (Al-kitab 'ala al-tawila.) - The book is on the table.
- سأذهب إلى السوق غداً. (Sa'athhab ila al-souq ghadan.) - I will go to the market tomorrow.
- الجو معتدل اليوم. (Al-jaww mu'tadil al-yawm.) - The weather is mild today.

# Sad (حزين)
- أشعر بالحزن الشديد. (Ash'ur bil-huzn al-shadeed.) - I feel very sad.
- لقد فقدت كل شيء. (Laqad faqadtu kulla shay'.) - I have lost everything.
- أتمنى لو كان الأمر مختلفاً. (Atamanna law kana al-amr mukhtalifan.) - I wish things were different.

# Surprised (متفاجئ)
- حقا؟! (Haqqan?!) - Really?!
- لم أكن أتوقع هذا! (Lam akun atawaqqa' hatha!) - I didn't expect this!
- يا إلهي! (Ya ilahi!) - Oh my God!

Alternatively: If you prefer to record your own audio, I've created the recording_scripts.txt file with sample sentences for each emotion.
To use your recordings, create a main data folder (e.g., my_audio_data).
Inside my_audio_data, create subfolders for each emotion exactly as named in the EMOTIONS dictionary in the notebook (e.g., Angry, Happy, Sad, etc.).
Save your .wav recordings into the corresponding emotion folder.
You will then need to update the DATA_PATH variable in the second code cell of the notebook to point to my_audio_data and potentially adjust the data loading code in the fourth cell if your file naming convention differs significantly from EAED's.
